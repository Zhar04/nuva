# Nuva — психологическая поддержка

Платформа: ИИ-подбор специалиста + маркетплейс психологов + анонимное сообщество.
Дизайн-шоукейс (iOS / Liquid Glass), бизнес-дашборд и стратегическая выжимка.

## Структура

| Файл | Что это |
|---|---|
| `Nuva.html` | Точка входа дизайн-шоукейса (React + Babel, открывается в браузере) |
| `nuva-showcase.jsx` | Галерея: бренд, дизайн-система, ключевые экраны |
| `nuva-screens-1.jsx` | Экраны: онбординг, ИИ-чат (интейк), главная |
| `nuva-tokens.jsx` | Дизайн-токены Liquid Glass, фрейм телефона, glass-примитивы, иконки |
| `nuva-brand.jsx` | Логотип (3 направления), иконка приложения, вордмарк |
| `nuva-i18n.jsx` | Локализация RU / EN / FR |
| `lib/ios-frame.jsx` | Фрейм устройства iOS 26 (статус-бар, клавиатура и т.д.) |
| `nuva-dashboard.html` | Автономный бизнес-дашборд: юнит-экономика, B2B ROI, маркетинг и MAU |
| `Nuvastrategy.pdf` | Стратегическая выжимка: продукт, рынок, экономика, B2B, маркетинг |

## Запуск

Шоукейс: открыть `Nuva.html` в браузере (нужен интернет — React/Babel грузятся с CDN).
Дашборд: открыть `nuva-dashboard.html` — без внешних зависимостей.
